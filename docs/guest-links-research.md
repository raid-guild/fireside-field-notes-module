# Guest Links Research

This tracks guest profile and session artifact enrichment for `content/encounters.json`.

## Source Priority

1. Portal event/session graph or Portal public post.
2. Guest-owned site or profile that links out to socials/projects.
3. Public project/company page that names the guest.
4. Search result snippets only when the identity match is specific and low-risk.

Do not add ambiguous socials to the app. Leave them here as candidates until confirmed.
